# Elevate Fitness
This project emphasizes a clean, performance driven user-experience with intuitive navigation and responsive layouts built to replicate a real world fitness brand experience. From a real time BMI calculator to dynamic class filtering by category and intensity, every feature was designed to make exploring fitness programs simple, engaging, and visually consistent across devices. The design prioritizes usability, quick navigation through React Router, and a mobile-first layout styled with Tailwind CSS.

# Key Features
- Built with React, React Router, and Tailwind CSS for a fully responsive, high-performance interface
- BMI calculator that provides real-time feedback for users tracking fitness goals
- Filterable class schedules with options to sort by any day of the week with a reminder of how many spots are currently left for that class
- Workout programs organized by intensity levels for an intuitive and personalized experience
- Smooth client-side navigation using React Router for seamless transitions between sections
- SEO-friendly structure and optimized images for improved performance and accessibility
